Архітектура

Структура серверів

Система складається з наступних додатків:

1) MQTT у якості Pub/Sub ABAC брокера;
2) LDAP для директорії користувачів;
3) DNS для безпеки іменного простору;
4) CA для CMS криптографії;
5) OCSP для перевірки сертифікатів;
6) TSP для підпису з позначкою часу;
7) CMP для видачі сертифікатів.
