Нова версія FORM

FORM - це генератор форм для різних бекендів, на якій написана сторінка депозитів pb.ua/depozit. Так, як я зараз оновлюю стек для ще одного промислового клієнта, вирішив випустити нову версію, якій вже буде 4 роки, після появи перших версій промислового використання.

— github.com/synrc/form — 4.3

Домашня сторінка проекту — form.n2o.dev є N2O додатком, яка використовує веб-фреймворк NITRO для оновлення DOM, та пересилає івенти по вебсокету і визначає цільовий формат генерації для бібліотеки FORM. Потім контрольні елементи NITRO перетворюються на HTML5 засобами самого NITRO, і після серіалізоване як рядок передається як керуюча команда для апдейта DOM на N2O JavaScript клієнта, який, довіряючи серверу, виконує тільки eval.

Додаток використовує статичні адреси JavaScript клієнта N2O, які розташовані на Github Pages, а самі форми беруться із джерел репозиторію бібліотеки і посилаються при встановленні вебсокет з'єднання в реакції повідомлення init протоколу NITRO.
